# 一些说明

## 如何启动

首先运行db.sql，主要作用就是创建名为wedb的数据库，也只要创建数据库就行了，不需要手动创建表，jpa会自动创建表；然后修改application.yml里的数据库用户名密码；然后就可以启动项目，第一次启动项目时会自动创建数据库表。
